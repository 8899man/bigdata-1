### 快捷方法
输入快捷方法后，再按TAB键即显示相关Java代码。

  Java代码            | 快捷方法        
----------------------|----------------
 System.out.println   |  sout           


另外，可通过`Live Templates`功能，构建相关快捷方式，如构建main方法：
```
public static void main(String[] args) {

}
``
